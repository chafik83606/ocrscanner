Demande de suppression des données — OCR Scanner

Éditeur : Dynaweb

Données concernées

OCR Scanner stocke les scans (images et texte extraits) uniquement sur votre appareil. Nous ne conservons pas vos documents sur nos serveurs.

Comment supprimer vos données

  1. Dans l'application (version Premium) : ouvrez l'historique, supprimez chaque scan (balayez ou bouton supprimer).
  2. Désinstallation : supprimez l'application OCR Scanner de votre téléphone. Les fichiers locaux de l'app sont alors effacés.
